# AutoRed
A GUI-based reconnaissance automation platform for penetration testers.

## Project
Final Year Project — APU, B.Sc. Cyber Security
Student: Hashreena Kaur (TP071939)

## Setup
1. Clone the repo
2. Run `python3 -m venv venv && source venv/bin/activate`
3. Run `pip install -r requirements.txt`

## Tools Required
- nmap, whatweb, ffuf (apt)
- subfinder, httpx (go install)
